# 数据获取指南 - hot-rank-sentiment-monitor

使用 AkShare 获取 A股热门股热度数据。

---

## 数据源选择

### 优先使用 AkShare 接口

热度/人气数据主要来自第三方平台（雪球、东方财富），理杏仁API暂无此类数据。本skill优先使用AkShare接口。

---

## 核心接口

### 1. 雪球热度排行榜

**接口**: `stock_hot_follow_xq`

**描述**: 雪球-沪深股市-热度排行榜-关注排行榜

**参数**:
- `symbol`: 选择"最热门"或"本周新增"

**返回字段**:
| 字段名 | 类型 | 说明 |
|--------|------|------|
| 股票代码 | object | 带市场前缀的代码（如SH600519） |
| 股票简称 | object | 股票名称 |
| 关注 | float64 | 累计关注数 |
| 最新价 | float64 | 当前股价（元） |

**使用示例**:
```python
import akshare as ak

# 获取最热门排行榜
df_hot = ak.stock_hot_follow_xq(symbol="最热门")
print(df_hot.head(20))

# 获取本周新增关注
df_new = ak.stock_hot_follow_xq(symbol="本周新增")
print(df_new.head(20))
```

**分析用途**:
- `最热门`: 评估长期热度水平（绝对关注度）
- `本周新增`: 计算ΔHot（热度变化），识别短期情绪转移

---

### 2. 东方财富人气飙升榜

**接口**: `stock_hot_up_em`

**描述**: 东方财富-个股人气榜-飙升榜

**参数**: 无

**返回字段**:
| 字段名 | 类型 | 说明 |
|--------|------|------|
| 排名较昨日变动 | int64 | 当日排名变化（正数=上升） |
| 当前排名 | int64 | 当前人气排名 |
| 代码 | object | 带市场前缀的代码 |
| 股票名称 | object | 股票名称 |
| 最新价 | float64 | 当前股价 |
| 涨跌额 | float64 | 当日涨跌额 |
| 涨跌幅 | float64 | 当日涨跌幅（%） |

**使用示例**:
```python
import akshare as ak

df_up = ak.stock_hot_up_em()
print(df_up.head(20))
```

**分析用途**:
- 识别短期情绪脉冲（排名变动剧烈=情绪驱动）
- 发现小盘题材股的"追涨"机会
- 监控过热/顶部风险（涨幅大+排名变动大）

### 3. 东方财富千股千评

**接口**: `stock_comment_em`

**描述**: 东方财富网-数据中心-特色数据-千股千评

**参数**: 无

**返回字段**:
| 字段名 | 类型 | 说明 |

| 市盈率 | float64 | 市盈率 |
| 主力成本 | float64 | 主力成本价 |
| 机构参与度 | float64 | 机构参与度（%） |
| 综合得分 | float64 | 综合评分 |
| 上升 | int64 | 排名上升位数 |
| 目前排名 | int64 | 当前排名 |

**使用示例**:
```python

```

**分析用途**:
- 关注指数：衡量市场注意力集中度
- 综合得分：结合多维度的评分
- 机构参与度：判断是否有机构背书

## 数据组合分析框架

### 1. 热度变化强度（ΔHot）计算

```python
import akshare as ak
import numpy as np

# 1. 获取本周新增数据
df_new = ak.stock_hot_follow_xq(symbol="本周新增")
df_hot = ak.stock_hot_follow_xq(symbol="最热门")

# 2. 合并计算ΔHot
merged = df_hot.merge(df_new, on='股票代码', suffixes=('_total', '_new'))
merged['ΔHot_占比%'] = merged['关注_new'] / merged['关注_total'] * 100

# 3. 计算分位数
percentiles = {
    'P90': np.percentile(merged['ΔHot_占比%'], 90),
    'P95': np.percentile(merged['ΔHot_占比%'], 95),
}

# 4. 筛选高ΔHot标的
high_delta = merged[merged['ΔHot_占比%'] > percentiles['P90']]
```

### 2. 情绪脉冲识别

```python
import akshare as ak

# 1. 获取飙升榜
df_up = ak.stock_hot_up_em()

# 2. 获取千股千评（含换手率）
df_comment = ak.stock_comment_em()

# 3. 合并数据计算情绪脉冲强度
# 情绪脉冲强度 = |涨跌幅| * 0.4 + 换手率 * 0.4 + (100-关注指数) * 0.2

# 4. 风险标签
# 高风险：涨跌幅>8% 且 换手率>10%
# 中风险：涨跌幅>5% 且 换手率>5%
```

### 3. 热度背离识别

```python
import akshare as ak

df = ak.stock_comment_em()

# 高关注但下跌（可能被错杀）
high_attention_low_return = df[
    (df['关注指数'] >= 90) & 
    (df['涨跌幅'] < -2)
]

# 低关注但上涨（可能被忽视）
low_attention_high_return = df[
    (df['关注指数'] < 70) & 
    (df['涨跌幅'] > 5)
]

# 相关性分析
corr = df['关注指数'].corr(df['涨跌幅'])
```

### 4. 资金与热度一致性验证

```python
import akshare as ak

# 获取热度数据
df_hot = ak.stock_hot_follow_xq(symbol="最热门")

# 获取资金流向
df_concept_flow = ak.stock_fund_flow_concept(symbol="即时")
df_sector_flow = ak.stock_fund_flow_industry(symbol="即时")

# 一致性判断
# 热度↑ + 资金↑ = 趋势确认
# 热度↑ + 资金↓ = 风险信号
```

## 关键指标汇总

| 指标名称 | 来源接口 | 计算方式 | 分析用途 |
|----------|----------|----------|----------|
| ΔHot占比% | stock_hot_follow_xq | 本周新增/累计关注*100 | 热度变化强度 |
| 关注指数 | stock_comment_em | 直接获取（0-100） | 市场注意力集中度 |
| 情绪脉冲强度 | stock_hot_up_em + stock_comment_em | \|涨跌幅\|*0.4+换手率*0.4+(100-关注指数)*0.2 | 短期情绪驱动风险 |
| 主力净流入 | stock_fund_flow_concept/industry | 直接获取 | 资金动向验证 |

## 注意事项

### 数据时效性
- 雪球数据：每日更新，反映T日收盘后状态
- 东方财富数据：盘中实时更新，T+0

### 数据口径差异
- 雪球"关注"为累计值，适合计算ΔHot
- 东方财富"关注指数"为标准化值（0-100），适合横截面对比
- 飙升榜"排名变动"为日频变化，需结合多日观察

### 常见陷阱
- 雪球数据需处理市场前缀（SH/SZ/BJ）
- 东方财富飙升榜可能包含新股/复牌股，需过滤
- 关注指数95+的标的可能已被充分定价
