# 历史压力测试情景定义 (A 股)

| 情景名称 | 时间段 | 背景 | 沪深 300 回撤 | 主要受损行业 |
| :--- | :--- | :--- | :--- | :--- |
| **2015 股灾** | 2015/06 - 2015/09 | 场外配资去杠杆，流动性危机 | -43% | TMT, 军工, 高估值成长 |
| **2018 贸易战** | 2018/01 - 2019/01 | 中美贸易摩擦, 信用收缩 | -32% | 出口型制造, 环保, 传媒 |
| **2021 核心资产泡沫** | 2021/02 - 2022/04 | 美债收益率上行, 消费/白马估值回归 | -35% | 食品饮料, 医药, 互联网 |
| **2024 流动性枯竭** | 2024/01 - 2024/02 | 雪球敲入, 量价背离 | -15% | 中小盘, 微盘股 |

### 辅助说明
- 测试组合时，应关注标的在这些时间段内的表现。
- 如标的未在对应时间段上市，使用行业基准作为替代。
