对A股公司执行完整估值流程。

## 执行步骤

1. 加载 `cn-data-source` skill，用理杏仁 API 获取以下数据：
   - 公司基本信息（名称、行业、财报类型）
   - 最近5年财务数据（收入、EBITDA、EBIT、净利润、现金流、资产负债）
   - 最新市场数据（股价、市值、PE、PB、股息率）
   - 同行业可比公司估值数据
   - 中国10年期国债收益率（无风险利率）

2. 将数据整理为标准输入格式（单位换算为百万元 CNY）

3. 加载 `company-valuation` skill，执行完整估值：
   - DCF 内在价值估值
   - 可比公司法（A股同行业）
   - EV/EBITDA、PE、PB 多倍数交叉验证
   - 情景分析（乐观/基准/悲观）
   - EV 到股权价值桥接
   - QA 质量检查

4. 输出中文估值报告，包含：
   - 估值区间与中值
   - 与当前股价的溢价/折价
   - 关键假设说明
   - 主要风险提示

## A股特殊处理

- 无风险利率：中国10年期国债收益率
- ERP：A股市场风险溢价 6%~8%
- 财务数据单位：元 → 百万元（÷1,000,000）
- 银行/保险/证券：自动识别财报类型，调整估值方法

## 示例

`/cn-company-valuation 600519`
`/cn-company-valuation 贵州茅台`
`/cn-company-valuation 300750`
