# Project Finance Model Essentials

Use for power, infrastructure, PPP, and contracted assets.

## Core Structure
- Separate construction and operations periods
- Cash flow based on CFADS
- Debt sizing constrained by DSCR

## Key Drivers
- Contracted price and escalation
- Opex and maintenance
- Capex profile and timing
- Tax and depreciation

## Outputs
- Cash flow and debt service schedule
- DSCR, LLCR, PLCR
- Equity IRR and NPV
- Sensitivity on price, volume, capex, debt terms
