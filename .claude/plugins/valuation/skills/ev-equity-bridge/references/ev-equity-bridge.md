# EV / Equity Bridge Essentials

## Standard Bridge Structure
- Enterprise Value (EV)
- Add: cash and non-operating assets
- Subtract: net debt
- Subtract: minority interest and preferreds
- Equals: Equity Value

## Dilution Handling
- Options, warrants, and converts adjusted via treasury stock or if-converted
- Report both basic and diluted equity value per share

## Outputs
- Bridge table (EV to Equity)
- Per-share values (basic and diluted)
- Key assumptions and sources
