# Valuation Formulas

## Core Definitions
- EV = Equity Value + Net Debt + Preferred + Minority Interest
- Equity Value = EV - Net Debt - Preferred - Minority Interest

## WACC
- WACC = Ke * E/(D+E) + Kd * (1-T) * D/(D+E)

## FCFF DCF
- FCFF = EBIT * (1-T) + D&A - Capex - Change in NWC
- PV = sum(FCFF / (1+WACC)^t) + Terminal Value

## Terminal Value
- Gordon: TV = FCFF_t+1 / (WACC - g)
- Exit multiple: TV = EBITDA * Multiple

## Per-Share
- Equity Value per share = Equity Value / Diluted shares
