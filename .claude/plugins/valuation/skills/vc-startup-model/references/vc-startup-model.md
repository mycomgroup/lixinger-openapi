# VC / Startup Valuation Notes

## Common Methods
- VC Method (exit multiple back-solve)
- First Chicago (scenario-weighted DCF)
- ARR or revenue multiples

## Key Metrics
- ARR, NRR, GRR, churn
- CAC, LTV, payback
- Burn rate, runway

## Outputs
- Valuation range (low/base/high)
- Key assumptions and sensitivities
- Exit assumptions and target return
