# Resource Valuation Essentials

## Methods
- rNPV (risk-adjusted NPV)
- NAV (asset revaluation)
- EV/Reserves, EV/Production

## Key Inputs
- Reserves by class (1P/2P/3P)
- Production profile and decline
- Commodity price deck and differentials
- Capex, opex, sustaining costs
- Tax, royalty, and reclamation costs

## Outputs
- NPV and rNPV
- IRR for single assets
- Price sensitivity matrix
