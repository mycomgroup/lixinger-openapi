## 2. 本次异常发现

按异常而不是按排名写结论，例如：
- `双腰斩后横盘极稳 + 突然放量但未破位` → 洗盘信号强
- `放量大阴线后 3 日缩量企稳` → 恐慌出清确认
- `基本面已转正但放量下跌` → 基本面背离型机会
- `放量破位创新低 + 基本面同步恶化` → 高风险回避

## 5. 未来跟踪信号

每只重点候选至少列出 2-3 个验证点：
- 放量大阴线后 5-10 个交易日是否缩量企稳
- 横盘区间下沿是否持续有效
- 是否出现放量阳线收复大阴线失地
- 后续成交量变化趋势（继续缩量 / 温和放量 / 再次异常放量）
- 下一个财报期基本面是否继续稳定或改善
- 大股东和机构持仓是否有变化
- 行业景气度是否触底回升

## 6. 风险提示

- 放量大阴线可能是洗盘也可能是真正破位，两者在发生时无法完全区分
- 双腰斩后横盘不等于一定会反转，可能继续阴跌
- 底部反转通常需要较长时间，不适合短线操作思维
- 基本面排雷能排除大部分风险但无法排除所有
- 本策略属于左侧策略，入选不等于立即买入信号
- 结论仅用于研究与筛选，不构成投资建议
