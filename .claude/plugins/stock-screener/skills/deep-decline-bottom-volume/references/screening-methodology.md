# 双腰斩横盘底部放量策略方法论

目标不是找所有超跌股票，而是找在底部形成扎实整理平台后出现异常量能信号的标的。

## 1. 硬筛选层

先排除明显不适合的公司：
- ST / 退市风险 / 黑名单
- 上市时间不足 3 年
- 持续亏损且负债率 > 80%
- 主业已消亡或被借壳
- 日均成交额极低（< 500 万元），流动性不足

## 2. 双腰斩判定

### 2.1 峰值识别

1. 回溯期默认 5-8 年，取期间后复权最高收盘价作为峰值
2. 若 5 年内最高价与 8 年内最高价差异不大（< 10%），取 5 年即可
3. 若峰值出现在上市初期（IPO 后 6 个月内），需谨慎对待，可能只是新股炒作

### 2.2 跌幅计算

```
累计跌幅 = (峰值价格 - 当前价格) / 峰值价格 × 100%
```

- ≥75% 视为满足双腰斩条件
- 75%-85%：标准深度下跌
- 85%-90%：极度深跌，需额外关注基本面是否崩塌
- >90%：高风险区域，优先排查退市和财务造假

### 2.3 下跌节奏验证

优先关注分阶段下跌而非一步到位的暴跌：
- 是否有两段明显的下跌波段（第一次腰斩 + 第二次腰斩）
- 两段下跌之间是否有反弹或横盘
- 连续暴跌（如退市前的连续跌停）不纳入本策略

## 3. 横盘震荡区间识别

### 3.1 核心判定

1. 在深度下跌后寻找持续 ≥18 个月的价格区间
2. 区间内 `(最高价 - 最低价) / 最低价 ≤ 30%`
3. 区间均价处于历史价格的底部 20% 分位以内
4. 区间内日平均成交量 ≤ 下跌阶段日均量的 50%（缩量确认）

### 3.2 均线粘合验证

- 20 日、60 日、120 日均线趋于收敛
- 均线之间的离散度 ≤ 5%
- 无明显的均线多头或空头排列

### 3.3 成交量萎缩确认

横盘期间的缩量是关键验证信号：
- 日均成交量达到近 3 年最低水平
- 或日均成交量处于历史 10% 分位以下
- 地量（极度萎缩）出现的频率增加

## 4. 底部放量大阴线检测

### 4.1 时间窗口

- 默认在近 30 个交易日内寻找信号
- 可扩展到近 60 个交易日（用户指定时）

### 4.2 放量判定

```
放量倍数 = 当日成交量 / MA(成交量, 20)
放量条件: 放量倍数 ≥ 2.0
```

注意：
- 计算 20 日均量时排除停牌日和涨跌停日
- 若近 20 日本身已有放量日，可用 60 日均量替代

### 4.3 大阴线判定

```
实体跌幅 = (开盘价 - 收盘价) / 开盘价 × 100%
大阴线条件: 实体跌幅 ≥ 4%
```

加分项：
- 实体占比高（上下影线短），说明卖压集中
- 跳空低开后继续下跌，恐慌情绪明显
- 但出现长下影线（下跌后被拉回），可能暗示承接力量

### 4.4 位置判定

```
底部区域 = 价格 ≤ 横盘区间均价 + (区间振幅 × 25%)
```

即大阴线收盘价应处于横盘区间的下半部分。

### 4.5 综合判分

| 条件 | 权重 | 说明 |
|---|---|---|
| 放量 ≥2x 均量 | 必要条件 | 量能异常是核心 |
| 跌幅 ≥4% | 必要条件 | 阴线幅度足够 |
| 处于横盘底部区域 | 必要条件 | 位置确认 |
| 放量后 3 日缩量企稳 | 加分 | 恐慌出清确认 |
| 长下影线 | 加分 | 承接力量存在 |
| 未破横盘区间下沿 | 加分 | 底部支撑有效 |
| 伴随利空消息 | 中性 | 利空出尽则为积极信号 |
| 放量后继续破位新低 | 扣分 | 可能是真正下跌 |

## 5. 基本面排雷层

技术信号发出后，必须进行基本面验证：

### 5.1 必查项目

| 项目 | 安全线 | 危险信号 |
|---|---|---|
| 最近一期营收增长 | ≥0% | 连续 3 期以上负增长 |
| 毛利率 | ≥15% | 毛利率持续下滑 |
| 净利润 | >0 | 连续亏损 |
| 经营现金流 | >0 | 持续为负 |
| 资产负债率 | ≤70% | >80% 且持续上升 |
| 商誉占净资产比 | ≤30% | >50% |

### 5.2 加分项

- 当前估值（PE、PB）处于历史低位
- 有股息率缓冲
- 行业景气度触底或出现改善迹象
- 大股东增持或回购

### 5.3 一票否决项

- ST 或退市警示
- 涉嫌财务造假或重大违规
- 大股东高比例质押 + 持续减持
- 主营业务已消亡

## 6. 机会分类框架

### `底部洗盘型`
- 放量大阴线后 1-3 日内缩量企稳
- 未有效跌破横盘区间下沿
- 横盘时间 ≥24 个月，筹码充分换手
- 基本面稳定或有改善迹象

### `恐慌出清型`
- 放量伴随明确利空消息（如业绩预亏、行业政策冲击）
- 利空属于一次性而非持续性
- 放量后利空不再发酵
- 价格迅速止跌

### `缩量横盘突变型`
- 放量前连续多日极度缩量（地量）
- 放量倍数 ≥3x 均量
- 异常显著但需进一步观察后续走势
- 可能是消息泄露或资金提前布局

### `基本面背离型`
- 基本面已有边际改善（营收转正、利润率回升）
- 但股价仍在底部区域放量下跌
- 属于市场认知滞后，中期看多
- 需要下一个财报期确认

### `高风险破位型`
- 放量大阴线后继续破位创新低
- 横盘区间被有效跌穿
- 基本面同步恶化
- 应标红回避，不纳入机会清单
