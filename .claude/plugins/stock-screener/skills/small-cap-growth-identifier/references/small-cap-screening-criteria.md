# 小盘成长策略方法论

目标是识别“小而强、小而新、小而早”的成长公司，并把真成长与伪成长分开。

## 1. 硬筛选层

先排除明显不适合继续研究的公司：
- `ST` / 退市 / 上市未满 3 年
- `PE-TTM(扣非) <= 0` 或 `PE-TTM <= 0`
- 扣非净利润 <= 0，或经营活动产生的现金流量净额 <= 0
- 总市值过小、流通市值过低，导致无法正常交易
- 近一年明显亏损且看不到修复证据
- 高商誉、高负债、财务质量明显失真

### 1.1 可交易性最低门槛

小盘成长默认比大盘价值更容易高估纸面机会，因此必须量化可交易性：

- 近 20 个交易日日均成交额 < `0.5` 亿元：直接淘汰
- 近 20 个交易日平均换手率 < `1%`：直接淘汰
- 若近 1 个交易日成交额 > 近 20 日日均成交额 `3` 倍，但此前成交长期偏弱，视为“单日放量幻觉”，不能按流动性达标处理
- 若最近 20 个交易日出现连续涨停，或累计 `3` 次及以上涨停 / 极端拥挤放量，只能降为观察名单，不能作为高优先机会
- 只有同时通过基本面硬门槛与可交易性门槛，才允许进入后续成长判断

## 2. 策略强化层

### 2.1 成长质量

重点看：
- 收入和利润是否同步增长
- 毛利率和 ROE 是否稳定或改善
- 研发投入是否支持未来增长
- 增长是不是主要来自低基数或并购并表

### 2.2 兑现证据

重点看：
- 财报里是否已经出现营收、利润、研发、盈利能力同步改善
- 互联互通持股或市场关注度是否开始变化
- 市场份额提升逻辑是否能从行业或产品定位得到支撑

### 2.3 被忽视原因

成长股被忽视往往不是因为没有故事，而是因为：
- 市值太小
- 行业偏冷
- 研究覆盖少
- 历史业绩一般但正在变好
- 板块整体估值压缩

### 2.4 伪成长识别

必须排查：
- 收入高增但利润率持续下滑
- 增长伴随应收、存货快速膨胀
- 研发投入不升反降，却试图讲高成长故事
- 资金面或股东结构恶化，说明成长兑现质量不高

## 3. 异常发现层

每次执行必须写出异常信号，例如：
- 收入和利润同步高增，但估值还未切换
- 研发投入明显抬升，利润释放可能滞后
- 收入增速很好，但毛利率和 ROE 没有跟上
- 市值很小、行业冷门，导致市场还没充分关注

## 4. 机会分类层

### `高质量小成长`

- 收入、利润、盈利质量较一致
- 有兑现证据
- 估值与成长匹配或仍有空间

### `困境修复型小盘成长`

- 过去有瑕疵，但关键经营指标开始修复
- 市场记忆仍停留在旧叙事

### `主题催化型成长`

- 业务存在主题催化
- 但仍需财报或经营数据验证，不可只看题材

### `伪成长预警`

- 高增长更多来自故事、低基数或财务失真
- 需要显式提醒回避或仅观察

## 5. 跟踪信号模板

对重点候选至少跟踪：
- 下个季度营收与利润是否继续同步兑现
- 毛利率、ROE、研发投入是否改善
- 互联互通持股是否开始变化
- 应收、存货、负债率是否恶化
- 是否出现新的客户、产品或覆盖催化
