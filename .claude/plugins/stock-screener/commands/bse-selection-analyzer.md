Load the `bse-selection-analyzer` skill.

默认目标：
- 找出业务强但关注度不足的北交所公司
- 识别流动性错杀、专精特新龙头与再评级线索
- 明确可交易性约束，避免只给出不可执行的名单

默认流程：
1. 先用 `lixinger-screener` 建北交所候选池。
2. 再评估流动性、估值、成长兑现、细分龙头属性与风险约束。
3. 强制输出 `本次异常发现`、`潜在机会清单`、`未来跟踪信号`。
4. 仅对入围股补充行情和公司信息，不默认全市场深拉。

如果用户没有给出条件，优先明确：
- 范围：`北证50` / `全北交所`
- 流动性门槛
- 结果数量
- 侧重点：`业绩兑现` / `流动性错杀` / `专精特新`

结果至少要分为：
- `流动性错杀`
- `专精特新龙头`
- `业绩兑现型`
- `北交所高风险样本`

Example:
`/bse-selection-analyzer 全北交所 流动性错杀优先`
