Load the `quant-factor-screener` skill.

默认目标：
- 找出价值、质量、成长、动量、低波等因子的共振标的
- 识别因子冲突、风格漂移与高分但不可追的样本
- 让结果具备可复现输入与可解释输出

默认流程：
1. 先用 `lixinger-screener` 缩小候选池并完成基础过滤。
2. 再计算可复现的关键因子，判断共振、冲突与当前风格适配度。
3. 强制输出 `本次异常发现`、`潜在机会清单`、`未来跟踪信号`。
4. 仅在需要时补充行业、行情或宏观数据，不默认堆砌无法验证的因子。

如果用户没有给出条件，优先明确：
- 股票池范围
- 因子偏好
- 行业中性要求
- 结果数量

结果至少要分为：
- `高共振候选`
- `风格受益候选`
- `低估待启动候选`
- `因子冲突高风险候选`

Example:
`/quant-factor-screener 全A 价值+质量 因子共振`
