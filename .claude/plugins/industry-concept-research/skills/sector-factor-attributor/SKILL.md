# 板块因子归因器

扮演量化基本面研究分析师。通过"三维收益归因拆解 + 风格因子暴露矩阵"框架，将行业涨跌幅分解为可解释的驱动来源，回答"板块涨了多少是靠估值扩张，多少是靠盈利改善"，增强配置逻辑的说服力。

---

## 工作流程

### 第一步：确认输入参数

确认以下内容：
1. **目标行业**：申万一级/二级行业名称（如"电子"、"半导体"）
2. **归因区间**：默认近3个月（`3m`），可选 `1m`、`6m`
3. **基准指数**：默认"沪深300"（用于超额收益计算）
4. **归因精度**：`full`（三维完整归因）/ `partial`（两维，估值+风险溢价，适用于盈利数据缺失时）

### 第二步：数据获取

从 `references/data-queries.md` 获取：

**步骤 2.1**：区间起止日期的行业 PE/PB 数据（用于估值变化计算）
```
API: cn/industry/fundamental/sw_2021
params: {date: "{start_date}", stockCodes: [行业代码], metricsList: ["pe_ttm.mcw","pb.mcw","mc"]}
params: {date: "{end_date}", stockCodes: [行业代码], metricsList: ["pe_ttm.mcw","pb.mcw","mc"]}
```

**步骤 2.2**：区间行业市值变化（用于计算总收益率）
```
通过起止日期的 mc 变化计算：R_total = (mc_end - mc_start) / mc_start
```

**步骤 2.3**（可选，如有盈利数据）：行业盈利变化数据
```
API: cn/industry/fs/sw_2021/hybrid
params: {date: "{quarter}", metricsList: ["q.ps.np.t","q.ps.np.t_y2y"]}
注：此API目前对部分行业不可用，缺数时触发降级（半归因）
```

**步骤 2.4**：宏观利率数据（用于风险溢价变化）
```
10年期国债收益率变化（区间起止）：手动输入或通过宏观数据源
```

### 第三步：三维收益归因拆解

**理论框架**：
```
总收益 = 估值扩张贡献 + 盈利贡献 + 风险溢价变化贡献 + 残差
```

#### 归因维度 1：估值扩张贡献

```
估值扩张贡献 = (PE_end / PE_start - 1) × 权重
或：PE_end × EPS_start / 初始市值 - 1
```

**解读**：
- 估值扩张贡献 > 50% → 本轮上涨主要是情绪/预期驱动，基本面支撑不足，警惕估值回归风险
- 估值扩张贡献 < 20% → 盈利驱动为主，上涨更具持续性

#### 归因维度 2：盈利（EPS）贡献

```
盈利贡献 = EPS增速 × 归因权重
近似：通过行业市值加权净利润同比增速估算
```

**降级方案（缺盈利数据时）**：
- 使用卖方一致盈利预期修正（定性估计：明显上修/基本不变/下修）
- 标注为"半归因"（PARTIAL），置信度下调15%

**解读**：
- EPS贡献 > 30% → 盈利改善是本轮上涨的实质支撑
- EPS贡献为负而价格仍涨 → 纯情绪/主题驱动，风险较高

#### 归因维度 3：风险溢价变化贡献

```
风险溢价变化 = 折现率变化对估值的影响
近似：利用10年国债收益率变化估算
若无利率数据：通过总收益 - 估值贡献 - 盈利贡献 = 残差（含风险溢价）
```

**解读**：
- 风险溢价大幅下降（利率下行 + 风险情绪好转）→ 利好估值扩张
- 风险溢价上升 → 即使基本面不差，也压制估值

**归因质量评级**：
- `FULL`：三维均有实际数据支撑
- `PARTIAL`：缺盈利数据，用两维归因（估值 + 残差近似风险溢价）
- `INDICATIVE`：缺多项数据，仅做定性方向判断

### 第四步：风格因子暴露评估

对目标行业评估以下风格因子暴露：

| 风格维度 | 判断指标 | 高暴露特征 |
|---------|---------|---------|
| **大盘/小盘** | 行业成分股平均市值 + 大市值占比 | 大盘因子：PE低、市值大、银行/消费 |
| **成长/价值** | PE vs PB相对高低 + 营收增速 | 成长因子：高PE、高营收增速、科技 |
| **质量/红利** | ROE、股息率、现金流稳定性 | 质量因子：高ROE稳定、股息率高 |
| **动量** | 近3M/6M涨跌幅分位 | 高动量：连续跑赢市场 |
| **低波动** | 历史收益率标准差 | 低波动：公用事业/消费 |

输出：各风格因子暴露方向（高暴露/中性/低暴露）+ 说明。

**风格因子解读意义**：
- 帮助判断行业是否受到"风格轮动"影响（如小盘风格盛行时小盘行业被动受益）
- 识别行业基本面改善 vs 纯风格因子溢价的区别

### 第五步：归因结论解读

综合三维归因，输出以下判断：

1. **本轮行情本质驱动**：情绪/估值驱动型 / 盈利支撑型 / 风险偏好驱动型 / 混合驱动
2. **持续性判断**：盈利支撑 = 高持续性；纯估值扩张 = 低持续性
3. **对 sector-rotation-detector 的增强**：为什么这个行业值得超配（或低配）？提供因子层面的解释

### 第六步：输出结构化结论

按 `references/output-template.md` 输出，必须包含：

1. **归因摘要**（一句话：本轮涨跌X%中，估值贡献Y%，盈利贡献Z%，风险溢价贡献W%）
2. **三维归因拆解表**（含各维度贡献比例和解读）
3. **风格因子暴露矩阵**
4. **归因质量**（FULL / PARTIAL / INDICATIVE）
5. **本轮行情本质驱动**（情绪/盈利/风险偏好/混合）
6. **配置建议增强**（为什么值得/不值得配置，因子层面的理由）
7. **置信度声明**（缺盈利数据时自动降为 PARTIAL 并标注）
8. **数据缺口说明**（如有）

## 降级策略

| 场景 | 降级方案 | 归因质量 | 置信度影响 |
|------|---------|---------|---------|
| 盈利（EPS）数据不可得 | 使用两维归因（估值+残差），标注 PARTIAL | PARTIAL | -15% |
| 利率数据不可得 | 用残差项整体替代风险溢价归因 | PARTIAL | -10% |
| 历史 PE 起始值不可得 | 用当前 PE 分位判断估值高低（定性） | INDICATIVE | -25% |
| 基准收益率不可得 | 省略超额收益分析，仅做绝对收益归因 | FULL/PARTIAL | -5% |

## 与其他 skill 的协同接口

- **→ sector-rotation-detector**：为轮动信号提供"为什么"的因子层面解释，增强配置逻辑

## 重要注意事项

- **A股盈利预期数据难获取**：卖方一致盈利预期数据通常需要第三方数据库，多数情况下只能做 PARTIAL 归因
- **估值扩张主导是 A 股常态**：A 股情绪驱动特征强，很多行情主要是估值扩张，非盈利驱动
- **风格因子在 A 股影响大**：2019-2021年大盘价值、2021-2022年小盘成长等风格轮动对行业配置影响显著，需单独评估

数据获取详见 `references/data-queries.md`，方法论详见 `references/methodology.md`。
