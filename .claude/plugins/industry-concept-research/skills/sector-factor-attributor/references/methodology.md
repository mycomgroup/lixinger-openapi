# 方法论 — sector-factor-attributor

本文档定义板块因子归因的核心算法、三维拆解公式和风格因子暴露评估框架。

---

## 1. 三维收益归因框架

### 1.1 理论基础

行业总收益率可以分解为：
```
R_total ≈ R_valuation + R_earnings + R_risk_premium + R_residual

其中：
- R_valuation：估值扩张/收缩贡献（PE/PB倍数变化）
- R_earnings：盈利增长贡献（EPS增速）
- R_risk_premium：风险溢价变化贡献（折现率变化）
- R_residual：残差（包含市场摩擦、流动性溢价等）
```

### 1.2 估值扩张贡献（维度1）

**完整公式**：
```python
# 如果持有EPS不变的假设：
R_valuation = (PE_end / PE_start) - 1

# 更精确的表达（市值直接法）：
R_total = (mc_end - mc_start) / mc_start

# 将总收益拆分：
# mc = PE × EPS × 股本  →  mc变化 = PE变化 × EPS变化（叠加效应）
# 一阶近似拆解：
R_valuation ≈ (PE_end - PE_start) / PE_start
R_earnings ≈ EPS_growth_rate  （即净利润增速）
```

**阈值解读**：
- `R_valuation > R_total × 60%`：估值扩张主导，情绪/预期驱动为主
- `R_valuation < R_total × 30%`：盈利或风险溢价变化主导
- `R_valuation < 0`（估值收缩但价格上涨）：盈利增速超越估值压缩，基本面极强

### 1.3 盈利贡献（维度2）

**完整公式**：
```python
R_earnings ≈ net_profit_yoy   # 净利润同比增速（行业层面）
# 或更准确：EPS增速 × 基期PE_start / PE_end
```

**数据降级**：
- 有净利润数据：使用 `q.ps.np.t_y2y`（净利润同比增速）
- 无净利润数据（多数行业）：
  - 方案A：使用卖方对行业盈利预期的定性修正方向（上修/不变/下修）
  - 方案B：仅用两维归因（估值 + 残差），触发 PARTIAL 归因模式

### 1.4 风险溢价变化贡献（维度3）

**近似计算**：
```python
# 方案1（有利率数据时）：
# 10年国债收益率变化 Δr → 通过 DDM 模型估算对 PE 的影响
# PE ≈ 1 / (r + risk_premium)  →  ΔPE ≈ -ΔPE / r²

# 方案2（无利率数据时，更实用）：
R_risk_premium ≈ R_total - R_valuation - R_earnings
# 将残差整体归入风险溢价 + 其他项
```

**残差归因解读**：
- 残差较大（>20%）且利率下行：大概率是风险溢价下降（风险偏好改善）推动
- 残差较大且无利率下行：可能是流动性溢价（北向资金流入、融资规模扩张等）

## 2. 风格因子暴露评估

### 2.1 五大风格因子判断规则

**大盘/小盘因子**：
```
大盘因子：行业内 Top10成分股市值合计 > 行业总市值 × 70%（龙头主导）
小盘因子：行业内 Top10成分股市值合计 < 行业总市值 × 40%（分散型）
典型大盘行业：银行、食品饮料、非银金融、家用电器
典型小盘行业：计算机软件细分、化工品细分
```

**成长/价值因子**：
```
成长因子：PE 10年分位数 > 60% 且 近年净利润增速 > 20%
价值因子：PE 10年分位数 < 40% 且 股息率 > 2.5%
典型成长行业：电子、计算机、医药研发
典型价值行业：银行、公用事业、交通运输
```

**质量/红利因子**：
```
高质量：ROE > 15% 且 ROE 稳定性高（历史波动率低）
高红利：股息率（dyr.mcw）> 3%
典型高质量高红利：食品饮料（茅台）、银行、家用电器
```

**动量因子**：
```
高动量：近3M涨跌幅 在全市场行业排名 Top 30%
低动量：近3M涨跌幅 在全市场行业排名 Bottom 30%
```

**低波动因子**：
```
低波动：历史日收益率标准差 < 市场均值 × 0.8
高波动：历史日收益率标准差 > 市场均值 × 1.3
典型低波动：银行、公用事业
典型高波动：计算机、传媒、军工
```

### 2.2 风格轮动背景判断

当行业受到风格因子驱动时，需判断当前市场是否处于对应风格的顺风期：
- 成长风格顺风期特征：科技主题活跃、利率下行、增量资金入市
- 价值风格顺风期特征：红利替代低利率存款、机构资金持续买入
- 小盘风格顺风期特征：活跃资金入市、T+0主题交易活跃

## 3. 归因质量评级规则

| 质量等级 | 条件 | 说明 |
|---------|------|------|
| `FULL` | 三维（估值+盈利+风险溢价）均有实际数据 | 完整归因，可信度高 |
| `PARTIAL` | 缺盈利数据，使用两维（估值+残差） | 盈利维度为定性近似 |
| `INDICATIVE` | 缺多项数据，仅做方向性判断 | 仅供参考 |

---

## 4. 信号规则（可测试规则）

**Rule 1（估值扩张主导 → 持续性弱）**：
```
IF {估值贡献 > 总收益 × 60% AND PE分位数 > 70%}
THEN {未来1-3个月，行业收益回撤概率 > 55%（均值回归压力）}
CONFIDENCE {0.55; 历史A股案例支撑}
FAILURE_MODE {盈利持续上修弥补估值压力; 流动性持续宽松}
```

**Rule 2（盈利主导 → 持续性强）**：
```
IF {盈利贡献 > 总收益 × 40% AND 盈利增速趋势持续改善}
THEN {未来1-2个季度，行业继续跑赢市场概率 > 60%}
CONFIDENCE {0.60; 基本面驱动规律}
FAILURE_MODE {盈利预期已过度定价（PE再度高估）; 宏观衰退导致盈利下修}
```

## 5. 降级策略详细说明

| 场景 | 触发条件 | 降级方法 | 归因质量 | 置信度扣减 |
|------|---------|---------|---------|---------|
| 无盈利数据 | `cn/industry/fs/sw_2021/hybrid` 无返回 | 使用卖方预期方向（上修/不变/下修）定性估计 | PARTIAL | -15% |
| 无历史 PE 起始值 | API 历史查询失败 | 用当前 PE 分位判断高低，不做变化量计算 | INDICATIVE | -25% |
| 无利率数据 | 无法获取10年国债收益率 | 用残差整体代表风险溢价项 | FULL/PARTIAL | -5% |
| 基准收益率不可得 | 无法获取沪深300区间涨跌幅 | 省略超额收益分析 | 不影响质量等级 | -5% |

## 6. A股特殊注意

- **A股PE扩张常态**：A股情绪驱动特征明显，估值扩张贡献通常大于盈利贡献（60%以上），这是A股特性而非分析错误
- **盈利预期透明度低**：卖方预期修正数据付费门槛高，多数分析只能做 PARTIAL 归因，这属于正常降级
- **政策驱动的"行政估值"**：国企/央企受到中特估主题驱动时，PE扩张不完全反映市场情绪，而是受估值修复政策影响，需单独标注
