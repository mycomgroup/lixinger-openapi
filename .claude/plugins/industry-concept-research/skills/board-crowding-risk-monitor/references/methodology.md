# 方法论 — board-crowding-risk-monitor

本文档定义板块拥挤度综合评分公式、五维信号体系和脆弱触发器定义。

---

## 1. 拥挤度综合评分公式

### 1.1 完整四维版（有资金集中度数据时）

```
crowding_score = 换手率分位子分 × 30%
              + 成交额占比子分 × 25%
              + 估值分位子分 × 25%
              + 资金集中度子分 × 20%
```

### 1.2 降级三维版（缺资金集中度数据时）

```
crowding_score_simple = 换手率分位子分 × 40%
                      + 成交额占比子分 × 35%
                      + 估值分位子分 × 25%
```

### 1.3 各维度子分满分标准

| 维度 | 满分（100）标准 | 零分（0）标准 |
|------|-------------|------------|
| 换手率分位 | 10年历史分位数 ≥ 90% | 历史分位数 = 0% |
| 成交额占比 | 占全A成交额的比例超过历史75分位的150% | 低于历史均值 |
| 估值分位 | PE 10年历史分位数 ≥ 90% | PE分位数 = 0% |
| 资金集中度 | 连续5日净流入且日均净流入加速+50%以上 | 持续净流出 |

## 2. 五类脆弱触发器定义与检验

### 触发器 1：量价背离（HIGH）

```
检验条件（AND）：
  A. 近5日成交额（ta的5日均值）相比5日前5日均值 下降 > 25%
  B. 近5日最大回撤 < 5%（价格仍在高位）

计算示例：
  成交额_5d_avg_now = mean(ta[-5:])
  成交额_5d_avg_before = mean(ta[-10:-5])
  volume_shrink = (成交额_5d_avg_now - 成交额_5d_avg_before) / 成交额_5d_avg_before
  
  if volume_shrink < -0.25 AND max_drawdown_5d < 0.05:
      trigger = TRUE (量价背离)
```

**经济含义**：成交量萎缩意味着新进资金不足以维持价格，价格虚高状态不可持续。

---

### 触发器 2：成交萎缩（MEDIUM）

```
检验条件：
  近3日成交额日均 < 近20日成交额日均 × 70%
  
计算示例：
  avg_3d = mean(ta[-3:])
  avg_20d = mean(ta[-20:])
  
  if avg_3d < avg_20d * 0.70:
      trigger = TRUE (成交萎缩)
```

**经济含义**：市场活跃度明显下降，主力资金开始退出。

---

### 触发器 3：龙头破位（HIGH）

```
检验条件（OR，任一满足即触发）：
  A. 板块市值最大的龙头股收盘价 < 20日移动均线
  B. 龙头股出现"高开低走"（今日最高价 > 昨收 × 1.02 但收盘价 < 昨收）连续2日
  
注：龙头股定义为行业成分股中市值最大的前1-2家公司
```

**经济含义**：龙头股领先反映资金撤退意愿，是板块行情逆转的最强先行信号。

---

### 触发器 4：涨停封板率下降（HIGH，仅概念板块）

```
检验条件：
  近3日涨停封板率（实际封板数 / 触板数） < 60%
  AND 较近10日峰值封板率下降 > 20 百分点
  
封板率近似计算（若无原始数据）：
  可通过"涨停家数 / 当日接近涨停（涨幅>9%）家数"近似
```

**经济含义**：封板率下降说明主力资金在高位主动出货，散户资金不足以维持封板。

---

### 触发器 5：北向/融资突然净流出（MEDIUM）

```
检验条件（AND）：
  A. 北向资金连续2日净卖出行业成分股（净卖出金额 > 1亿）
  B. 融资余额连续2日下降

注：若数据不可得，跳过此触发器，在 data_gaps 中标注
```

**经济含义**：外资和杠杆资金同时撤出是市场脆弱性的综合信号。

## 3. 操作建议决策树

```
if crowding_score >= 90 OR triggered_HIGH_count >= 2:
    recommendation = "AVOID"  # 清仓或回避
    
elif crowding_score >= 80 OR (crowding_score >= 70 AND triggered_HIGH_count >= 1):
    recommendation = "REDUCE"  # 减仓至半仓以下
    
elif crowding_score >= 60 OR triggered_total_count >= 2:
    recommendation = "WATCH"  # 控仓，不加仓，密切监控
    
else:
    recommendation = "NORMAL"  # 可正常参与
```

## 4. 失效条件与风险解除规则

拥挤度高 → 建议降仓后，以下情况可重新评估（风险解除信号）：

| 解除信号 | 条件 |
|---------|------|
| 拥挤度自然回落 | 综合评分连续3日下降，降至60以下 |
| 成交量回暖 + 价格回升 | 成交额恢复至20日均值以上 + 价格创新高 |
| 龙头股重新站上均线 | 龙头股收盘价重回20日均线以上并放量 |
| 估值回落至合理区间 | PE 10年分位数回落至70%以下 |

## 5. 信号规则（可测试规则）

**Rule 1（高拥挤度 + 量价背离 → 短期回调概率高）**：
```
IF {crowding_score >= 80 AND 量价背离触发}
THEN {未来5-10个交易日，板块回调 > 10% 的概率 >= 60%}
CONFIDENCE {0.60; 历史极端拥挤案例（如2020年医药、2021年新能源）支撑}
FAILURE_MODE {新的强力催化剂出现（如超预期政策）维持行情; 板块平台整理消化拥挤度}
```

**Rule 2（龙头破位 → 板块回撤信号）**：
```
IF {龙头破位触发 AND crowding_score >= 70}
THEN {未来3-7个交易日，板块回撤 > 8% 的概率 >= 55%}
CONFIDENCE {0.55; 初步估计}
FAILURE_MODE {龙头股假破位（翌日收回均线以上）; 资金切换到板块内其他个股}
```

## 6. A股特殊注意

- **T+1制度加剧拥挤风险**：高位买入后无法当日卖出，T+1下行情逆转时止损成本更高
- **涨停板制度创造"封住拥挤度"假象**：日内封住涨停板时成交量被压制，拥挤度信号可能低估
- **概念板块拥挤度半衰期短**：概念板块从高拥挤到退潮的时间通常只有3-10个交易日，比行业板块（数周到数月）快很多
- **龙头破位的A股特殊性**：A股个股可能连续涨停后突然三连板，龙头破位后跌幅往往是普通市场的2-3倍
